SuperX Browser Agent Privacy Policy

Last updated: May 11, 2026

SuperX Browser Agent connects the local SuperX desktop app with the user's Chrome browser to provide browser assistance requested by the user. This policy applies to the SuperX Browser Agent Chrome extension.

Single Purpose

The extension's single purpose is to connect Chrome to the local SuperX desktop app so SuperX can inspect and operate browser pages for user-requested browser assistance tasks.

Data Accessed

The extension requests access to all websites so browser assistance can work consistently across sites selected by the user. Depending on the task requested in SuperX, the extension may access:

  • Tab information, including page URL, page title, tab ID, window ID, and tab state.
  • Website content in the active browser context, including visible text, links, forms, page structure, actionable elements, media state, and visible-tab screenshots.
  • User-requested page actions, such as clicks, form input, option selection, scrolling, navigation, and media-control results.

How Data Is Used

Data is used only to provide the extension's single purpose: browser assistance through the local SuperX desktop app. The extension accesses browser or page data only when SuperX sends an explicit command for a user-requested task.

Examples include reading page structure so SuperX can answer a question about the current page, filling a form field when requested by the user, clicking a visible page control, scrolling the page, or capturing a visible-tab screenshot for a user-requested task.

Data Sharing

SuperX Browser Agent does not sell user data, does not use user data for advertising, and does not share user data with third-party analytics, advertising networks, or data brokers.

Local App Communication

Browser and page data may be sent to the local SuperX desktop app installed on the user's device through Chrome Native Messaging to complete user-requested tasks.

The extension itself does not transmit browser data to an external server. Any processing beyond the extension happens through the local SuperX desktop app and the services or accounts the user configures there.

Data Not Accessed

The extension does not request Chrome permissions for cookies, passwords, bookmarks, downloads, or the Chrome history API. It does not read cookie values, saved passwords, payment information, or browser bookmarks.

Remote Code

The extension does not load or execute remote JavaScript or WebAssembly code. Page operations are performed by code packaged with the extension.

Chrome API Limited Use

The use of information received from Chrome APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
